# Stopwatch Web Application

A simple and interactive stopwatch web application that allows users to measure time intervals accurately.

## 🚀 Features

- Start timer
- Pause timer
- Reset timer
- Lap time tracking
- Responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (setInterval, Event Handling)

## 📂 Project Structure

├── index.html  
├── style.css  
└── script.js  

## 📌 Functionality

- Uses `setInterval()` to update time every second.
- Tracks minutes, seconds, and milliseconds.
- Dynamically updates the DOM.
- Stores lap times in a list.

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in a browser

## 🎯 Learning Outcome

- Working with time-based functions
- Event listeners
- DOM updates in real-time
- State management
